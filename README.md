# 2026_to_learn
Survey topics for 2026 learning
    
## Python 程式基礎與軟體工程:    
[learn-python-the-smart-way-v2](https://github.com/datawhalechina/learn-python-the-smart-way-v2/)	快速複習 Python 基礎語法與進階特性，可作為常備工具書。    
[writing_solid_python_code_gitbook](https://github.com/L1nwatch/writing_solid_python_code_gitbook)	學習編寫高質量、易讀且易維護的 Python 程式碼規範，建立良好工程習慣。    
[effective_python_notebook](https://github.com/kingli74/effective_python_notebook)	掌握 Python 進階技巧與開發最佳實踐，從「寫得出」邁向「寫得好」。    
[office-automation](https://github.com/datawhalechina/office-automation)	運用 Python 進行日常辦公文件自動化處理，提升工作效率。    
[handy-n8n](https://github.com/datawhalechina/handy-n8n  )	學習輕量級 n8n 工作流自動化工具的操作與應用。    

## 數學與演算法基石:    
[hello-algo](https://github.com/krahets/hello-algo)	透過豐富的動畫與圖解，直觀學習基礎資料結構與演算法思維。    
[math-for-ai](https://github.com/datawhalechina/math-for-ai)	建立 AI 與機器學習必備的微積分、線性代數與機率統計基礎。    
[intro-mathmodel](https://github.com/datawhalechina/intro-mathmodel)	學習如何將真實世界的複雜問題，抽象轉化為數學模型來求解。    
[numerical-linear-algebra-v2](https://github.com/kingli74/numerical-linear-algebra-v2)	深入探討線性代數在數值計算與機器學習底層中的實際應用。    
[scientific-computing](https://github.com/datawhalechina/scientific-computing)	熟練運用 NumPy 與 SciPy 等函式庫進行高效能的科學與矩陣計算。    
    
## 資料庫與數據分析:    
[wonderful-sql](https://github.com/datawhalechina/wonderful-sql)	學習資料庫查詢語言 SQL，掌握從資料庫撈取與初步處理資料的必備技能。    
[joyful-pandas](https://github.com/datawhalechina/joyful-pandas)	深入解析 pandas 核心語法與資料操作技巧，強化表格型資料處理能力。    
[hands-on-data-analysis](https://github.com/datawhalechina/hands-on-data-analysis)	透過 Kaggle 等實戰專案，完整走過資料清理、特徵工程與分析流程。    

## 資料視覺化:    
[fantastic-matplotlib](https://github.com/datawhalechina/fantastic-matplotlib)	系統化學習 Matplotlib 繪製各類靜態圖表，打穩視覺化底層基礎。    
[viz-mastery](https://github.com/datawhalechina/viz-mastery/)	掌握資料視覺化的美學設計原則與進階排版技巧，提升圖表說服力。    
[Panel (HoloViz 生態系)](https://panel.holoviz.org/)  非常適合處理大型數據，能與 pandas 與 hvPlot 這類強大的資料視覺化庫無縫整合，是打造複雜資料儀表板的絕佳利器。    
    
## 傳統機器學習 (ML):    
[pumpkin-book](https://github.com/datawhalechina/pumpkin-book)	搭配周志華《機器學習》（西瓜書），對核心數學推導與公式進行詳細解析。    
[statistical-learning-method-solutions-manual](https://github.com/datawhalechina/statistical-learning-method-solutions-manual)	李航《統計學習方法》的演算法推導筆記與課後習題解答。    
[machine-learning-toy-code](https://github.com/datawhalechina/machine-learning-toy-code)	透過從頭刻寫程式碼的方式，深入理解分類、迴歸等經典演算法的底層邏輯。    

## 深度學習與應用 (DL):    
[leedl-tutorial](https://github.com/datawhalechina/leedl-tutorial)	台大李宏毅教授深度學習課程的實戰筆記，是銜接理論與實作的最佳入門。    
[thorough-pytorch](https://github.com/datawhalechina/thorough-pytorch) 	系統性學習目前業界主流的 PyTorch 深度學習框架操作與網路建構。    
[dive-into-cv-pytorch](https://github.com/datawhalechina/dive-into-cv-pytorch)	結合 PyTorch 學習電腦視覺 (CV) 領域的核心技術，如影像分類與物件偵測。    
[whale-paper-pal](https://github.com/datawhalechina/whale-paper-pal)	學習閱讀 AI 領域學術論文的拆解方法與輔助工具，培養前沿技術追蹤能力。    
[《Dive into Deep Learning》](https://d2l.ai/)  這是一本可以「直接執行」的互動式電子書。它把每一個深度學習的數學公式，都直接用 PyTorch 程式碼寫出來讓你對照，非常適合拿來打通數學理論與程式實作的任督二脈。    

## 系統部署與進階擴充:    
[docker-notes](https://github.com/datawhalechina/docker-notes)	學習 Docker 容器化技術，為未來將訓練好的模型進行部署封裝做準備。    
[wow-fullstack](https://github.com/datawhalechina/wow-fullstack)	了解 AI 模型如何結合前端介面與後端架構，開發完整的全端應用服務。    
[whale-quant](https://github.com/datawhalechina/whale-quant)	探索 Python 在量化金融與投資策略上的跨領域應用實作。

[Gemini API Developer Guide](https://ai.google.dev/docs/prompt_best_practices)  官方開發者文件，包含許多適合軟體工程的提示詞範例
